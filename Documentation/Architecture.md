# AI-Augmented SOC Architecture

## Architecture Components

1. Splunk
   - Centralized log collection and monitoring platform.

2. Zeek
   - Network traffic monitoring and analysis tool.

3. VirusTotal
   - Threat intelligence enrichment source.

4. Shuffle SOAR
   - Security orchestration and automated response platform.

5. Analyst Dashboard
   - Provides visibility into alerts, incidents, and investigations.

## Workflow

1. Network and system logs are collected.
2. Logs are forwarded to Splunk.
3. Suspicious events trigger alerts.
4. Indicators are enriched using VirusTotal.
5. Shuffle SOAR automates investigation and response actions.
6. Analysts review incidents and take final actions.

## Benefits

- Faster threat detection
- Reduced analyst workload
- Automated incident response
- Improved visibility
- Centralized security operations
