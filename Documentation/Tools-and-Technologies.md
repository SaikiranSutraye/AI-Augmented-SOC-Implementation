# Tools and Technologies

## Splunk Enterprise

Used for centralized log collection, monitoring, searching, and security event analysis.

## Zeek

Used for network traffic monitoring and generating detailed network security logs.

## VirusTotal

Used for threat intelligence enrichment and reputation analysis of suspicious indicators.

## Shuffle SOAR

Used for security orchestration, automation, and incident response workflows.

## Ubuntu Linux

Used as the primary SOC monitoring platform.

## Windows Server 2022

Used as the target system for log generation and testing activities.

## VirtualBox

Used to host and manage virtual machines within the lab environment.

## Security Concepts Applied

- Log Analysis
- Threat Intelligence
- Network Monitoring
- Incident Response
- Security Automation
- Alert Correlation
- SOC Operations
