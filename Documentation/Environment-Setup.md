# Environment Setup

## Lab Environment

The project was implemented in a virtualized cybersecurity lab environment.

## Components Used

- Ubuntu Linux
- Windows Server 2022
- Splunk Enterprise
- Zeek Network Security Monitor
- Shuffle SOAR
- VirusTotal
- VirtualBox

## System Configuration

### Ubuntu Machine

- Operating System: Ubuntu Linux
- Role: Security Monitoring and SOC Services

### Windows Server

- Operating System: Windows Server 2022
- Role: Log Generation and Testing Environment

## Network Configuration

- Internal virtual network communication
- Log forwarding enabled
- Security monitoring traffic captured through Zeek

## Purpose

The environment simulates a real-world Security Operations Center (SOC) for threat detection, investigation, enrichment, and response automation.
