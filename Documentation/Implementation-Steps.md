# Implementation Steps

## Step 1 - Environment Preparation

- Installed Ubuntu Linux and Windows Server 2022 virtual machines.
- Configured networking between virtual machines.
- Verified connectivity and system readiness.

## Step 2 - Splunk Deployment

- Installed Splunk Enterprise.
- Configured data inputs and log collection.
- Verified successful log ingestion.

## Step 3 - Zeek Integration

- Installed Zeek Network Security Monitor.
- Configured network traffic monitoring.
- Generated and analyzed network logs.

## Step 4 - VirusTotal Integration

- Configured VirusTotal API integration.
- Enriched Indicators of Compromise (IOCs).
- Automated reputation checking for suspicious indicators.

## Step 5 - Shuffle SOAR Deployment

- Installed and configured Shuffle SOAR.
- Created automated workflows for alert handling.
- Integrated external security tools.

## Step 6 - Alert Investigation Workflow

- Generated test security events.
- Collected and analyzed alerts.
- Automated enrichment and response actions.

## Step 7 - Validation and Testing

- Tested end-to-end workflow.
- Verified alert generation and response.
- Confirmed successful integration of all components.
